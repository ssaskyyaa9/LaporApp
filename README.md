# LaporApp
Platform pengaduan masyarakat berbasis website dan aplikasi mobile untuk melaporkan masalah di lingkungan sekitar dengan mudah dan real-time.

# Fitur Utama
- **Laporan Instan:** Masyarakat bisa melaporkan masalah lingkungan langsung dari lokasi.
- **Multi-Platform:** Tersedia dalam versi website dan aplikasi mobile yang mudah diakses.
- **Sistem Real-Time:** Pengiriman dan pemrosesan data laporan dilakukan secara langsung.

### Teknologi yang Digunakan
Proyek ini dibangun menggunakan susunan teknologi modern:
- **Web Frontend:** Next.js
- **Mobile App:** React Native (Expo)
- **Backend API:** Express.js dan Node.js
- **Bahasa:** JavaScript
